# hodotaev_praktika - Подсистема работы с партнерами

## Описание
Приложение для управления партнерами производственной компании «Мастер Пол».

## Требования
- PostgreSQL 14 или выше
- .NET 8.0 SDK
- Windows 10/11

## Установка базы данных

### Вариант 1: Через pgAdmin
1. Откройте pgAdmin и подключитесь к PostgreSQL
2. Откройте Query Tool
3. Скопируйте содержимое файла `database/hodotaev_praktika.sql`
4. Выполните скрипт (F5 или кнопка Execute)

### Вариант 2: Через командную строку (psql)
```bash
psql -U postgres -f database/hodotaev_praktika.sql
```

### Параметры подключения
- **Имя БД:** hodotaev_praktika
- **Роль:** app
- **Пароль:** 123456789
- **Схема:** app
- **Хост:** localhost
- **Порт:** 5432

## Запуск приложения

### Вариант 1: Через Visual Studio 2022
1. Откройте файл `hodotaev_praktika.sln`
2. Нажмите F5 или кнопку "Start"

### Вариант 2: Через командную строку
```bash
cd hodotaev_wpf
dotnet run
```

### Вариант 3: Запуск исполняемого файла
```bash
.\publish\hodotaev_wpf.exe
```

## Структура проекта
```
hodotaev_praktika/
├── database/                      # Скрипты БД
│   └── hodotaev_praktika.sql     # Скрипт создания БД
├── hodotaev_library/              # Библиотека классов
│   ├── Models/                   # Модели данных
│   ├── Data/                     # DbContext
│   └── Services/                 # Бизнес-логика
├── hodotaev_wpf/                  # WPF приложение
│   ├── MainWindow.xaml           # Главное окно
│   ├── PartnerEditWindow.xaml    # Окно редактирования
│   └── SalesHistoryWindow.xaml   # Окно истории продаж
├── hodotaev_library.Tests/        # Модульные тесты
├── publish/                       # Опубликованное приложение
└── hodotaev_praktika.sln         # Решение Visual Studio
```

## Функционал
- ✅ Просмотр списка партнеров
- ✅ Добавление нового партнера
- ✅ Редактирование данных партнера
- ✅ Удаление партнера (с проверкой наличия продаж)
- ✅ Расчет индивидуальной скидки
- ✅ Просмотр истории продаж партнера
- ✅ Валидация данных
- ✅ Обработка исключений

## Правила расчета скидки
| Объем продаж (руб.) | Скидка |
|---------------------|--------|
| до 10 000           | 0%     |
| 10 000 - 50 000     | 5%     |
| 50 000 - 300 000    | 10%    |
| более 300 000       | 15%    |
